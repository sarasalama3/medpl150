Blink Animation

Puppet Warp Animation

Reflection

For this assignment, I created two animated GIFs using Illustrator and Photoshop. The first GIF is a blink animation based on the vector eye I previously made in Illustrator. I created multiple versions of the eye in different stages of closing and opening, then brought them into Photoshop and arranged them frame by frame in the timeline. I adjusted the frame timing so the blink would feel smooth and natural.

For the second GIF, I used a photo of myself and created a cut-out figure with Select Subject and a layer mask. I duplicated the figure into multiple layers and used Puppet Warp to make small changes in pose, especially in the head and arm position, so the movement would feel subtle and realistic. Then I turned the layers into frames, set the animation to loop forever, and exported it as a GIF. This project helped me understand how animation works through small frame-by-frame changes and how Photoshop tools can create simple motion from still images.
